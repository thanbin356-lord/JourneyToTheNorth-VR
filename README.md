# Journey To The North (VR)

Journey To The North is a small VR exploration project inspired by Viking villages and northern landscapes.
Players can freely walk around the environment, explore the village, and interact with NPCs to learn more about Viking culture and daily life.

This project was created to improve my skills in:

* Environment creation
* Environment optimization for VR
* Lighting and post-processing
* Level design
* Creating interactive 3D NPCs
* VR interaction and immersion

The main focus of this project was building a believable atmosphere and creating a relaxing exploration experience inside a VR environment.

## Technologies Used

* Unity Engine
* C#
* VR SDK
* Blender

## Project Purpose

This is a personal learning project made to practice VR development and improve my environment art and level design skills.
